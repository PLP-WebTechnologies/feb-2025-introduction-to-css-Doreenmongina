CSS (Cascading Style Sheets) is a stylesheet language used for describing the presentation of a document written in HTML or XML. It controls the layout of multiple web pages all at once.

CSS is a cornerstone technology of the World Wide Web, alongside HTML and JavaScript.

CSS is designed to enable the separation of presentation and content, including layout, colors, and fonts.

CSS can be applied to HTML documents in three ways: inline styles, internal stylesheets, and external stylesheets.
